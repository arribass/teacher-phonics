# Teacher Phonics 🍎

**Teacher Phonics** es una aplicación web interactiva diseñada para que docentes y educadores puedan crear materiales personalizados de fonética (phonics) en inglés para sus alumnos.

Por ahora, la aplicación permite que cada usuario genere su propio abecedario de fonemas personalizado y pueda guardarlo para usarlo en sus clases. Esto sirve como base para diseñar tarjetas de sonido (flashcards), estructurar lecturas adaptadas al aula o preparar hojas de ejercicios.
